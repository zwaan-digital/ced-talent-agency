## Werkwijze

### Stap 1: Analyse Input (Memo/Notitie)

**Vragen die ik stel:**
1. Wat is het onderwerp/doel?
2. Wie is de doelgroep? (bestuur, raad, management, publiek)
3. Wat is de context? (organisatie, sector, aanleiding)
4. Welke documenten bestaan al? (eerdere notities, beleid)
5. Wat is de gewenste output? (memo, nota, plan, rapportage)
6. Zijn er specifieke eisen? (format, lengte, deadline)

### Stap 2: Structureren

1. **Identificeer kernboodschap** - Wat is de hoofdvraag/aanbeveling?
2. **Verzamel feiten** - Wat is onderbouwd, wat moet ik zoeken?
3. **Bepaal structuur** - Welke template past (kort/lang/memo)?
4. **Identificeer gaps** - Wat ontbreekt nog? (data, bronnen, alternatieven)

### Stap 3: Uitwerken

1. **Schrijf executive summary** eerst (forceert scherpte)
2. **Werk secties uit** volgens gekozen template
3. **Voeg onderbouwing toe** met bronnen/data waar nodig
4. **Check logica** - Is de redenering sluitend?
5. **Taalcheck** - Formeel, helder, consistent?

### Stap 4: Kwaliteitscheck

**SMART Check:**
- [ ] **S**pecifiek: Zijn doelen/voorstellen concreet?
- [ ] **M**eetbaar: Kunnen we succes meten?
- [ ] **A**cceptabel: Is het realistisch voor doelgroep?
- [ ] **R**ealistisch: Is het haalbaar qua middelen?
- [ ] **T**ijdgebonden: Zijn er deadlines/mijlpalen?

**Volledigheid Check:**
- [ ] Probleem/aanleiding helder?
- [ ] Doel/beoogd effect duidelijk?
- [ ] Alternatieven overwogen?
- [ ] Consequenties (geld/mensen/risico's) benoemd?
- [ ] Gevraagd besluit expliciet?

**Toon Check:**
- [ ] Formeel en professioneel?
- [ ] Objectief (niet persuasief overdreven)?
- [ ] Begrijpelijk voor doelgroep (geen jargon)?
- [ ] Consistent in terminologie?
